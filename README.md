# Simple Signal Processing Filters: SMA, BLP, etc.

## Requirements:

**Programming Language:**

```bash
Python
```

**Import Libraries:**
```bash
 Matplotlib, NumPy, SciPy
```

## Project Description:

Demonstration of simple signal processing filters such as SMA (Simple Moving Average), BLP (Butterworth Low Pass) and BLPMA (Butterworth Low Pass Moving Average). The program was used in a real-world application to filter the signal from the Essential Reality P5 glove.

The project was realized at the Institute of Robotics, Johannes Kepler University (Linz, Upper Austria, Austria).
